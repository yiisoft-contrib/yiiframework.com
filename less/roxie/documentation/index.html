3.1 ColorSwitcher

It is plugin developed by ourselfs. It allows you to change a color in real time. Basically you can change the whole appereance by yourself. Below you can see hexadecimal codes of color we used.

Now let’s take a look how to set your favourite look.

Setup color

  1. At first you have to decided which color is best for you.
  2. Open .html page in some editor.
  3. To change your color find line 55.

Now just simply change class=“color-green“ to the one you choosen on the start of this procedure e.g class=“color-red“ now save this change and refres edited page and enjoy.

Remove ColorSwitcher

When you are done with color changes is no longer necessary to have ColorSwither on your page.

  1. Open .html files and delete everything between lines 68 - 91 including. On the image below are the lines which you delete. Now the icon of ColorSwitcher on the page is gone.
  2. Delete file color-switcher.css
  3. Delete file color-switcher.js

3.2 Font Awesome

It’s one of the most powerfull tools. Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS. Let’s take a look. In the index below the carousel section you can find four icons in the circles.

This icon of mobile, pencil, trophy and gear is the font awesome so let’s say we wanna change them to more suitable for your purposes. Open the index.html in some editor and find the line number 428

Here you can change appearance of this icon just simply remove class „fa-mobile“ and type for instance „fa-money“. Now save the changes and refresh the index.html page and icon is different.

For full documentation how to customize this plugin (size, color, rotation etc.) and full list of different types of icons go to www.fontawesome.io

3.9 jQuery CountTo

This plugin allows us visualize counting to some number. You can see this plugin on pages index-application and index-product or on the pictures below.

You can easily customizing it by following this steps.

  1. Open .html e.g. index-application.
  2. Go to line 460
  3. Edit atributes data-from, data-to, data-refresh-interval to values that fits to your purposes.
  4. If you want to changes word "clients" to more suitable go to line 461 and just edit it.

Full documentation you can check here github.com/mhuggins/jquery-countTo

3.10 Range slider jQuery UI

With this tool we can easily set some range. We use this to selecting price on eshop sidebar.

You can easily customizing it by following this steps.

  1. Open custom.js file
  2. Go to line 54 to set up minimum (just edit the red number)
  3. Go to line 55 to set up maximum (just edit the red number)

Full documentation you can check here http://jqueryui.com/slider

4. FAQ

If I buy an Single application license, can I remove your Copyright-Line?

Yes, in any kind of license you definitely can remove copyright. But if you like our work you can place there just "created by pixelized" it is completely up to you.

How do i connect contact form with my email.

This is only html5, css3 & jQuery theme. There isn't any script to do it. You have to use some external script probably PHP or something. We are selling many of copies and everybody has slightly different requirement so we are not able to build single one script.

Will you build PHP script to connect contact form to my email.

No, this isn't our field of expertise.

Can i use, images included in this theme?

In credits section you can find links to every image. So find image you want in credits and check licence on source page.
